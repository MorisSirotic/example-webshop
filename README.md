# example-webshop
Available operations in the application:
  1) Creation of the user account/Registration
  2) Login
  3) Account verification through email
  4) Password reset option
  5) Browsing though products
  6) Searching though products
  7) Displaying products according to some criteria(price, alphabetical order)
  8) Adding products to the cart
  9) Deleting products from the cart
  10) Order creation
  11) Browsing through orders 
  12) Order notification through email
  13) Verification before each request being sent to see if the JWT token is expired or incorrect
  14) Automatic logout whenever the JWT token expires or becomes incorrect



Izvodljive operacije u aplikaciji:
  1) Izradba korisničkog računa/Registracija
  2) Prijava
  3) Potrvda računa putem e-maila
  4) Opcija resetiranje lozinke
  5) Pregledavanje proizvoda
  6) Pretraživanje proizvoda(search)
  7) Prikaz proizovda po nekojoj kriteriji(cijena, redoslijed)
  8) Dodavanje proizvoda u korpu
  9) Brisanje proizvoda iz korpe
  10) Izradba narudžbe
  11) Pregled narudžbi
  12) Obaviješaj o narudžbi putem e-maila
  13) Provjera prije svakog requesta da li je JWT token istekao ili je neispravan
  14) Automatska odjava kada JWT token istekne ili postane neispravan
